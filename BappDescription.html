This extension passively reports detailed server error messages.

Detailed error messages are often not visible during the normal course of testing. Some examples are:

  • Java: "[SEVERE] at net.minecraft.server.World.tickEntities(World.java:1146)"
  • ASP.Net: "System.Web.UI.Page.ProcessRequestMain(Boolean includeStagesBeforeAsyncPoint) +2071"
  • PHP: "c() called at [/tmp/include.php:10]"
  • Perl: "Use of uninitialized value in string eq at /Library/Perl/5.8.6/WWW/Mechanize.pm line 695"

New in version 2.0.0:

  • Reports error messages detected in any HTTP response
  • Configurable selection of Burp tools to scan

Source: https://github.com/augustd/burp-suite-error-message-checks
